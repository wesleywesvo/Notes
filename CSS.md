## Parent and Child Elements

### Can separate elements of HTML

```html
<style>
	div {
		color: blue;
		font-family: cursive;
	}

	ul {
		color: red;
	}
</style>

<div>
	<h2>Grocery List</h2>
	<h4>- the three c's</h4>

	<ul>
		<li>Coffee</li>
		<li>Cheese</li>
		<li>Chips</li>
	</ul>
</div>
```
